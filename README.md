# FINTECH 533 Breakout Strategy

This project develops, backtests, and documents a walk-forward breakout trading strategy using historical ETF price data fetched with ShinyBroker.

The final deliverable is designed for publication through GitHub Pages and includes:

- A clearly documented breakout detection function
- A full trade blotter available as downloadable CSV
- Trade outcome analysis for successful, timed-out, and stop-loss exits
- Performance metrics such as average return per trade and annualized Sharpe ratio
- A professional write-up explaining the methodology in plain English
